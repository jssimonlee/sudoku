# 프리미엄 수도쿠 웹 게임

순수 HTML, CSS, JavaScript로 만든 모던 반응형 수도쿠 게임입니다.  
라이트/다크 모드, 글래스모피즘 UI, 다양한 편의 기능을 지원합니다.

## 주요 기능 ✨

- **프리미엄 UI** – 라이트/다크 모드 토글, 글래스모피즘 효과, 부드러운 애니메이션
- **스마트 퍼즐 생성** – 논리만으로 풀 수 있는 퍼즐만 출제 (찍기 최소화), 유일 정답 보장
- **3단계 난이도** – 쉬움 / 보통 / 어려움
- **실시간 검증** – 오류 즉시 감지, 흔들림 애니메이션 & 실수 카운트 (3회 초과 시 게임 오버)
- **하이라이트** – 선택 셀의 행·열·박스·같은 숫자 자동 강조
- **숫자 완성 연출** – 숫자 9개 완성 시 에메랄드 그린 캐스케이드 파도 & 팡파레 효과
- **게임 클리어 연출** – 1~9번 순서 전체 보드 캐스케이드 후 결과 팝업
- **녹색 완성 표시 토글 (`#`)** – 완성된 9개 셀과 숫자 패드를 녹색 강조 ON/OFF
- **메모 모드 (Notes ✏️)** – 빈 칸에 예상 숫자 1~9 기입, 재클릭 시 삭제, 정답 확정 시 관련 셀 자동 제거
- **번호 선택 & 빠른 입력** – 번호 하나를 선택해두고 빈 칸 더블클릭 또는 **F / ` / ~ / Space / Esc 키**로 빠르게 입력
- **일시정지 (Pause ⏸)** – 보드를 가리고 타이머 정지, 중앙 ▶ 또는 **일시정지 상태에서만 Space 키**로 재개
- **실행 취소 (Undo)** – 이전 입력으로 되돌리기
- **반응형** – 모바일·데스크톱 모두 최적화

## 조작 방법 🎮

`index.html`을 브라우저에서 바로 열어 플레이 (별도 설치 불필요)

| 동작 | 방법 |
|---|---|
| 숫자 입력 | 셀 클릭 → 화면 숫자 패드 또는 키보드 1~9 |
| 번호 선택 | 숫자 패드 클릭 / 키보드 1~9 / 보드 안 숫자 클릭 |
| 선택 번호 빠른 입력 | 빈 칸 더블클릭 또는 **F / ` / ~ / Space / Esc 키** |
| 메모 입력 | Notes ✏️ 활성화 후 숫자 클릭 (토글) |
| 지우기 | Erase 버튼 또는 Backspace / Delete |
| 실행 취소 | Undo 버튼 또는 Ctrl+Z / Cmd+Z |
| 방향 이동 | 방향키 ↑↓←→ |
| 일시정지 / 재개 | ⏸ 버튼, 또는 일시정지 상태에서만 Space |
| 녹색 강조 전환 | `#` 버튼 (헤더 우상단) |
| 테마 전환 | 🌙 / ☀ 버튼 (헤더 우상단) |

## 게임 규칙

- 각 행, 열, 3×3 박스에 1~9가 하나씩만 들어가야 합니다.
- 오류 3회 발생 시 게임 오버.

## 기술 스택 🛠️

- **프론트엔드**: HTML5, CSS3, Vanilla JavaScript (의존성 없음)
- **아이콘**: FontAwesome 6 (CDN)
- **폰트**: Google Fonts – Outfit (CDN)
